# Jogo de Adivinhação

![](https://imgur.com/aVe2vKL.gif)

## Introdução

Jogo de adivinhação, onde o computador pensará em um número, e você, jogador, precisará adivinhá-lo. 

## Funcionalidades

- A cada erro, a máquina lhe dirá se o número chutado foi maior ou menor do que o pensado.
- Você também poderá escolher o nível de dificuldade do jogo, e isso lhe dará mais ou menos oportunidades de chutar um número. Ao final, se você ganhar, o computador lhe dirá quantos pontos você fez, baseando-se em quão bons eram seus chutes.
